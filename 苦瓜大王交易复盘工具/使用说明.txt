## 📱 如何访问与安装

## 🧭 页面导航（底部 6 个 Tab）

打开工具后底部有 6 个导航按钮，点击即可切换页面：

### 📊 仪表盘（首页）
交易复盘的总控中心，打开工具第一眼看到的就是这里。

**展示内容：**
- 总交易笔数、胜率（%）、总盈亏金额
- 最大盈利 / 最大亏损 / 平均盈亏
- 连胜/连败次数
- A–D 评级和情绪分析
- 月度盈亏趋势折线图
- 最近交易记录快捷预览

**首次使用：** 如果还没有交易数据，仪表盘会显示新手引导卡片，点击按钮即可开始。

---

### 📋 交易（记账本）

**添加一笔交易：**
1. 点击右上角 **「+ 新建」** 按钮
2. 填写交易信息：
   - 交易对（如 BTCUSDT）
   - 方向：做多 / 做空
   - 开仓价格、平仓价格
   - 数量（张数或币数）
   - 手续费
   - 交易所（Binance / OKX / 其他）
   - 产品类型（永续合约 / 交割合约 / 现货 / 期权）
   - 复盘状态（待复盘 / 已复盘 / 盈利 / 亏损）
   - 备注
3. 点击保存

**筛选与搜索：**
- 按品种、方向、结果、交易所、产品类型、复盘状态快速筛选
- 按日期范围过滤
- 支持关键词搜索

**编辑 / 删除：** 每条记录右侧有编辑 ✏️ 和删除 🗑️ 按钮。

---

### 📅 日历视图

按日期查看每日盈亏情况：
- 有交易的日期会标记颜色（红色 = 盈利，蓝色 = 亏损）
- 点击日期查看当日所有交易明细
- 顶栏可切换月份
- 底部显示月盈亏合计和交易天数

---

### 📊 AI 智能市场分析

AI 分析需要自行配置 API Key（DeepSeek 或 OpenAI），在设置页填写。

**市场分析功能：**
1. 切换到「分析」页面
2. 点击右上角 **「🔄 刷新分析」**
3. 系统自动拉取币安实时行情 + 恐惧贪婪指数
4. AI 生成解读：
   - 📊 大盘概况 — 当前市场状态概览
   - 📈 走势预测 — 短期走势判断
   - ⚡ 异动监测 — 异常波动的币种
   - 🎯 机会提示 — 潜在交易机会
   - 📋 具体点位与建议 — 支撑/阻力位及操作建议
5. 分析记录保留在下方「分析历史」中

**截图分析功能：**
- 拖拽 K 线截图到上传区域，或 Ctrl+V 粘贴
- 填写交易对和时间框架（如 BTCUSDT 15分钟图）
- 选择分析框架（ICT / 价格行为 / 缠论等）
- AI 自动识别图表并给出技术分析

> 小提示：没有 API Key 也能看到市场数据（币安价格、恐慌贪婪指数），只是 AI 解读部分不可用。

### ⚙️ 设置

**外观与货币：**
- 🌓 深色模式 / 浅色模式切换
- 💰 货币单位 USD ↔ CNY 切换

**数据管理：**
| 功能 | 说明 |
|---|---|
| 📥 导出数据（JSON） | 一键备份所有交易记录到文件 |
| 📤 导入数据（JSON） | 从备份文件恢复数据 |
| 🗑️ 清除所有数据 | 清空全部记录（操作不可撤销！） |

**CSV 批量导入：**
支持从交易所直接导出的 CSV 文件批量导入：
1. 选择导入格式（自动检测 / Binance 持仓历史 / 统一交易明细 / 通用格式）
2. 点击上传区域或拖拽文件
3. 系统自动解析并导入所有交易记录
4. 下方调试区域会显示导入详情

**AI 分析设置：**
| 选项 | 说明 |
|---|---|
| API 类型 | DeepSeek / OpenAI / 自定义（Custom） |
| API Key | 填入你的 Key |
| 自定义 API URL | 使用自己的中转或私有部署时填写 |

## 🔒 数据安全说明

| 项目 | 说明 |
|---|---|
| 数据存储位置 | 仅保存在你的浏览器本地（localStorage）|
| 数据上传 | 零数据上传，不经过任何服务器 |
| 数据备份 | 设置页 → 导出 JSON 备份到本地文件 |
| 数据恢复 | 新设备上访问网站 → 设置页 → 导入 JSON |
| 换设备 | 旧设备导出 → 新设备导入，数据即可迁移 |
| 清缓存前 | 务必先导出备份，否则数据会丢失 |

## ❓ 常见问题

**Q: AI 分析按钮点不了/没反应？**
A: 去设置页检查是否已填入 API Key，以及 API 类型是否选对（支持 DeepSeek 和 OpenAI 兼容接口）。

**Q: CSV 导入报错怎么办？**
A: 上传区域下方会显示详细调试信息，根据提示调整 CSV 格式。尝试选择对应的导入格式。

**Q: 数据会丢吗？**
A: 正常使用不会丢，但建议定期导出 JSON 备份。清除浏览器缓存或换设备前务必先备份。

**Q: 能同时用 DeepSeek 和 OpenAI 吗？**
A: 一次只能选一种 API 类型，切换后点击「刷新分析」即可使用新 Key。

**Q: 手机屏幕显示不全？**
A: 此工具为移动端优化设计，安装到主屏幕效果最佳。横屏使用建议切换桌面模式。

**Q: 可以离线使用吗？**
A: 可以。安装 PWA 后，已加载的数据和页面可在离线状态下查看。但实时行情和 AI 分析需要网络。
