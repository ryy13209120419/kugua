📌 什么是这个工具？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

这是一款交易复盘工具，帮你记录和分析加密货币/美股/外汇等交易。
核心功能：
  ✅ 手动记录交易（品种、方向、价格、盈亏）
  ✅ 截图导入：上传交易截图，自动识别并导入为待复盘状态
  ✅ AI 智能分析：接入 DeepSeek，分析你的交易质量
  ✅ 实时行情（服务器模式）：查看大盘数据、AI 市场分析
  ✅ 多设备：手机和电脑均可使用
  ✅ 数据本地存储，不上传任何服务器


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🚀 快速开始
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

两种使用模式：

【模式一：纯网页模式（推荐新手）】
  1. 直接双击 index.html
  2. 在浏览器中打开即可使用
  3. 支持手动记账、截图导入、AI分析
  4. 数据保存在浏览器本地
  ⚠️ 此模式无需服务器，但无法获取实时行情

【模式二：服务器模式（完整功能）】
  1. 首先运行 setup.bat（仅首次，安装 Python 依赖）
  2. 然后运行 start.bat
  3. 在浏览器打开 http://localhost:5000
  4. 可获取实时行情、CoinGlass 数据、AI 市场分析
  5. 多个设备连接同一台电脑也可同步数据


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 基本操作 - 记录交易
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开工具，点击底部「交易」
2. 点击「➕ 新建交易」
3. 填写品种（如 BTCUSDT）、方向（做多/做空）
4. 输入入场价、出场价、数量
5. 输入盈亏（正数为盈利，负数为亏损）
6. 选择交易所、产品类型
7. 填写交易信号、情绪、错误（可选）
8. 点击保存

提示：你可以随时点击已有的交易记录进行修改或删除。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📸 截图导入交易（核心功能）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 在 Binance/OKX 等交易所截图你的仓位记录
2. 把截图传到电脑上
3. 打开工具，点击底部「分析」
4. 找到「📸 截图导入交易记录」区域
5. 点击上传截图
6. 点击「🔍 识别并导入」
7. 工具自动识别：品种、方向、价格、盈亏
8. 确认后保存为「待复盘」状态

💡 手机使用技巧：
  - 在手机上截图 → 微信发文件到电脑 → 上传识别
  - 如果你在电脑上截图，直接 Ctrl+V 粘贴即可

⚠️ OCR 识别需要网络加载 Tesseract.js 库
  如识别失败，会自动切换到手动输入表单

🤖 AI 智能分析（交易复盘）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AI 功能需要配置 DeepSeek API Key：

1. 打开工具，点击底部「设置」
2. 找到「🤖 AI 分析设置」
3. 在 API Key 输入框中填写你的 DeepSeek API Key
4. 点击「保存」
5. 在交易列表中，点击某笔交易的「🤖 AI 分析」按钮
6. AI 会自动分析这笔交易的质量并给出建议

💡 DeepSeek API Key 获取：
  1. 访问 https://platform.deepseek.com/
  2. 注册账号 → 创建 API Key
  3. 充值少量金额（约 10 元可用很久）

📊 实时行情与市场分析
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

需要服务器模式（运行 start.bat 或 start_server.py）：

1. 确保服务器已启动
2. 打开工具，点击底部「分析」
3. 点击「🔄 刷新分析」
4. 大盘概况板块将显示：总市值、BTC Dominance、主流币价格等
5. 如配置了 DeepSeek API Key，AI 会生成完整市场分析
  包括：走势预测、异动监测、机会提示、具体点位建议
6. 每次分析自动保存到「分析历史」

方法一：离线模式
  1. 把「苦瓜大王交易复盘工具」文件夹传到手机上
  2. 用手机浏览器打开 index.html
  3. 所有功能正常使用（截图导入、手动记账等）
  4. 数据保存在手机浏览器的本地存储中
  ⚠️ 不支持实时行情（需要服务器）

方法二：通过服务器访问
  1. 在电脑上运行 start.bat
  2. 确保手机和电脑在同一 WiFi 下
  3. 手机浏览器访问 http://电脑IP:5000
  4. 所有功能完整可用

❓ 常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: 打开后页面无法点击？
A: 按 Ctrl+F5 强制刷新页面。如果仍然不行，尝试清除浏览器缓存。

Q: 数据存在哪里？
A: 所有数据存储在浏览器本地 (localStorage)，不上传任何服务器。
   清除浏览器缓存会丢失数据，建议定期导出备份。

Q: 如何备份数据？
A: 在「设置」页面 → 数据管理 → 点击「导出」按钮
   备份文件是 JSON 格式，需要恢复时点击「导入」即可。

Q: 截图导入不识别？
A: 确保截图清晰，包含品种、方向、价格等关键信息。
   如果 OCR 识别失败，会自动弹出表单供手动输入。

Q: AI 分析提示"API Key 未配置"？
A: 在「设置」页的 AI 分析设置中填写 DeepSeek API Key 并保存。

Q: 行情数据不显示？
A: 确保已运行 start.bat 启动服务器，然后按「刷新分析」。

Q: 手机和电脑数据能同步吗？
A: 可以在电脑运行服务器，手机在同一 WiFi 下访问即可同步。
   如果离线使用，数据分别保存在各设备的浏览器中，不会同步。
